# 議題研究 (Topic Research)

> 針對特定議題進行深度、跨領域、系統性的探討與分析

## 什麼是議題研究？

議題研究是一種聚焦於特定問題或主題的研究方法，不限於單一公司、不受特定時間區間限制，旨在深入理解複雜、開放性的問題。類似於探討「Hard Questions」或「Wicked Problems」。

## 與其他研究類型的區別

| 特性 | 議題研究 | 企業研究 | 一般文章 |
|------|---------|---------|---------|
| 研究對象 | 特定議題/問題 | 特定公司 | 事件/技術/經驗 |
| 時間範圍 | 不限定，視議題而定 | 當前+歷史 | 通常較短期 |
| 分析深度 | 深度/系統性 | 深度 | 中度 |
| 涵蓋範圍 | 可能很廣（跨領域） | 聚焦單一企業 | 聚焦單一主題 |
| 結論類型 | 多元視角/情境式 | 評估/建議 | 資訊/觀點 |

## 適用情境

- **政策議題**：如：「AI 監管應該如何平衡創新與風險？」
- **科技倫理**：如：「自動駕駛的道德困境如何解決？」
- **社會趨勢**：如：「遠距工作將如何改變城市發展？」
- **產業變革**：如：「電動車轉型對傳統車廠供應鏈的衝擊？」
- **哲學探討**：如：「數位時代的隱私權該如何定義？」
- **跨域整合**：如：「氣候變遷與經濟發展的取捨？」

## 研究框架概覽

議題研究採用十部分的系統性框架：

```
┌─────────────────────────────────────────────────────────┐
│                    執行摘要                              │
│              (核心發現 & 立場聲明)                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Part 1: 議題界定          Part 2: 歷史脈絡              │
│  (問題陳述/研究邊界)       (起源/演進/典範轉移)           │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Part 3: 利害關係人        Part 4: 多維度分析            │
│  (權力/利益/衝突)          (PESTEL/系統/倫理/經濟)        │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Part 5: 論證分析          Part 6: 比較分析              │
│  (各方立場/證據評估)       (跨國/歷史/跨領域)             │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Part 7: 情境規劃          Part 8: 解決方案              │
│  (未來情境/預警信號)       (政策工具/行動建議)            │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Part 9: 研究限制          Part 10: 研究結論             │
│  (偏見自覺/未解問題)       (綜合評估/立場陳述)            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 採用的核心方法論

| 方法 | 來源 | 用途 |
|------|------|------|
| 系統性文獻回顧 | 學術研究標準 | 建立知識基礎 |
| PESTEL 分析 | 策略管理 | 宏觀環境掃描 |
| 利害關係人分析 | R. Edward Freeman | 理解權力結構 |
| 系統思考 | Donella Meadows / Peter Senge | 複雜因果分析 |
| 情境規劃 | Shell / Peter Schwartz | 未來可能性探索 |
| 論證分析 | Stephen Toulmin | 邏輯與證據評估 |
| 倫理分析 | 多元倫理學傳統 | 價值判斷檢視 |
| 比較研究 | 政治學/社會學 | 跨國跨域學習 |

## 研究品質指標

優秀的議題研究應具備：

- [ ] **問題界定清晰**：研究範圍明確，不過度發散
- [ ] **多元視角呈現**：公正呈現各方立場，避免偏頗
- [ ] **證據品質標註**：區分事實、推論與價值判斷
- [ ] **系統性分析**：理解因果結構與反饋機制
- [ ] **承認不確定性**：對結論的信心程度誠實標註
- [ ] **實用的結論**：提供可操作的行動建議
- [ ] **自我反思**：揭露研究限制與可能偏見


## 模板版本選擇

議題研究提供兩種模板版本，根據研究需求選擇：

### Standard 版本（完整版）
- **模板位置：** `/templates/topic-research-template.md`
- **適用情境：** 深度學術研究、系統性政策分析
- **結構：** 10 部分完整框架
- **目標字數：** 15,000-50,000 字
- **完成時間：** 4-8 週

### Lite 版本（簡化版）
- **模板位置：** `/templates/topic-research-lite-template.md`
- **適用情境：** 快節奏議題探討、初步研究
- **結構：** 5 部分精簡框架
- **目標字數：** 3,000-8,000 字
- **完成時間：** 1-2 週

### 如何選擇？

```
時間預算
├── 1-2 週 → Lite 版本
└── 4 週以上 → Standard 版本

議題複雜度
├── 簡單/中等 → Lite 版本
└── 複雜/跨領域 → Standard 版本

研究目的
├── 初探/快速理解 → Lite 版本
├── 決策支援 → Lite 或 Standard（視時間）
└── 學術發表/深度分析 → Standard 版本
```
