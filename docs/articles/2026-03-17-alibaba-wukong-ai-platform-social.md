# 把釘釘打碎，用 AI 重建：阿里巴巴「悟空」要讓 8 億使用者的工作方式徹底改變

> 釘釘 CEO 陳航說：「過去是人用釘釘來工作，未來是 AI 用釘釘來工作。」這句話背後，是阿里巴巴剛剛用組織重組、產品重寫、硬體佈局三管齊下交出的答卷。

## 版本 1：完整版（Facebook / LinkedIn）

3 月 16 日深夜，阿里巴巴宣布成立全新的 Alibaba Token Hub（ATH）事業群，由集團 CEO 吳泳銘直接負責。隔天，釘釘 AI 2.0 年度新品發布會上，釘釘創始人兼 CEO 陳航站上舞台，說了一句讓整個科技圈都在轉發的話：「今天把釘釘打碎，用 AI 重建，煉出『悟空』。」

這場發布會的核心產品「悟空」，是阿里巴巴定義的「全球首個企業級 AI Agent 平台」。它能夠 24 小時自主工作，扮演的是真正意義上的 AI 同事角色，而非聊天機器人或文件助手。

---

### 組織重組：Token 就是新的生產力

先說背景。ATH 事業群的成立，代表阿里巴巴正式將 AI 戰略從「技術研發」推進到「商業落地」。ATH 的核心使命是「創造 Token、輸送 Token、應用 Token」，旗下整合了通義實驗室、MaaS 業務線、千問事業部，以及全新成立的悟空事業部和 AI 創新事業部。

這個架構調整清楚表明，阿里巴巴不再只是做大模型，而是要把大模型的能力變成企業可以直接消費的「Token」。就像電力公司不只發電，還要架設輸電網路、建立計費系統，讓每一度電都能送到使用者手中。

---

### 悟空：24 小時自主工作的 AI 原生工作平台

悟空是一款獨立應用程式，即日起開啟邀請測試，同時也將直接內建到超過 2,000 萬家企業組織使用的釘釘之中。

為了讓悟空真正能「工作」而不只是「對話」，釘釘做了一件大膽的事：重寫底層程式碼，進行全面 CLI（命令列介面）化改造。這意味著悟空 Agent 可以原生操作釘釘上千項功能，包括建立文件、排程會議、管理審批流程、查詢資料，而不是像傳統 AI 助手那樣模擬人類點擊圖形介面。

這個技術決策的意義在於，當 AI Agent 透過 CLI 操作系統，它的執行速度更快、錯誤率更低、可組合性更強。這跟 Anthropic 的 Claude Computer Use 走的是完全不同的路線。Claude 試圖讓 AI 學會像人類一樣操作螢幕，釘釘則是把整個系統改造成 AI 可以直接「說話」的語言。

使用者可以在電腦或手機上遠端喚起悟空，交代一個複雜任務，然後去做別的事。悟空會自主規劃步驟、呼叫釘釘的各項能力、協調多個 AI Agent 分工合作，最終交付結果。

---

### OPT：一人團隊，十大行業

發布會上最引人注目的概念是 OPT（One Person Team，一人團隊）。阿里巴巴同步發布了十大行業解決方案，首批覆蓋電商、跨境電商、知識型部落客、軟體開發、門店經營、設計、製造、法律、財稅、獵頭。

OPT 的核心理念很簡單，一個人加上一群 AI Agent，就能完成過去需要一整個團隊才能處理的工作。這已超越效率提升的層次，進入工作型態根本性重構的境界。一個跨境電商賣家不再需要分別聘請客服、翻譯、資料分析師，悟空平台上的 AI Agent 可以同時扮演這些角色。

悟空還同步推出了 AI 能力市場（Skill Marketplace），全面相容開源 Skill 體系。這表示第三方開發者可以為悟空生態開發各種專業技能，企業也可以根據自身需求組合不同的 AI 能力。

---

### AI 硬體：從軟體到實體的延伸

悟空不只是軟體平台，發布會上阿里巴巴也同步展示了 AI 硬體佈局。

釘釘此前已推出 AI 硬體「釘釘 A1」，一張卡片大小的錄音裝置，可以貼在手機背面，支援錄音、語音轉文字、即時翻譯、會議紀錄自動生成。這款產品上線後登上天貓和抖音的「錄音筆暢銷榜」榜首。

這次發布會上，新一代 A1 Pro 錄音卡和 Cleer H1 AI 耳機正式亮相。更重要的是企業級 AI 硬體「釘釘 Real」。如果說 A1 是 AI 釘釘的「耳朵」，那釘釘 Real 就是它的「身體」。這款硬體具備實名身份驗證、企業內網資料連接、即時資料監控等功能，本質上是 AI Agent 在物理世界中的執行終端。

釘釘 Real 解決的是企業 AI 落地的最後一哩路：當 AI Agent 需要讀取企業內部資料、即時處理資訊、與物理環境互動時，它需要一個安全可控的實體裝置。在中國三大智慧辦公平台（釘釘、企業微信、飛書）中，釘釘是唯一採用「軟硬體整合」策略的公司。

---

### Token 計費：像用水電一樣用 AI

悟空採用了一個值得關注的商業模式。企業不需要支付月費或年費，而是根據 AI 實際生成或處理的文字、程式碼、資料量來計費，就像水電費帳單一樣，用多少付多少。

這個 Token 計費模式和 ATH 事業群的命名遙相呼應，也解釋了為什麼阿里巴巴要把新事業群叫做「Token Hub」。在阿里巴巴的戰略藍圖中，Token 不只是 AI 模型的技術術語，更是未來企業 AI 消費的基本計量單位。

---

### 全球化野心

悟空不打算只留在中國。陳航在發布會上宣布，悟空將同步進軍全球市場，後續支持連接微信、Slack 等全球主流即時通訊平台。

加上阿里巴巴集團旗下淘寶、天貓、支付寶、阿里雲等 B 端商家能力正逐步以 Skill 形式接入悟空，這個平台正在成為阿里巴巴 AI 能力在企業工作場景的統一出口。

---

### 競爭格局：企業 AI Agent 大戰開打

悟空的發布並非孤立事件。就在同一時期，騰訊推出了 WorkBuddy 和 QClaw 等 Agent 工具，NVIDIA 在 GTC 大會上發布了 Agent Toolkit，而開源社群的 OpenClaw 框架也引發了廣泛關注。阿里巴巴的差異化在於，它擁有中國最大的企業協作平台（釘釘，超過 2,000 萬企業組織、8 億使用者），有完整的電商和雲端服務生態可以接入，還有自研的千問大模型作為底層支撐。悟空站在一個龐大的企業服務帝國之上，起跑點就與其他競爭者截然不同。

然而，這場 AI Agent 平台戰不只是中國市場的局部競賽，全球科技巨頭幾乎同時在佈局類似的產品線。

---

### 從通路角度看：誰能成為 AI Agent 的「作業系統」？

當每家巨頭都在推出類 OpenClaw 的 Agent 框架時，真正的競爭焦點已經從「誰的模型更強」轉移到「誰能掌握 Agent 的通路與分發」。這場戰爭的本質，是爭奪 AI 時代的作業系統地位。

**Microsoft 和 Google 的路線：既有通路的 AI 化。** Microsoft 把 Copilot 深度嵌入 365 生態系，讓 AI Agent 直接在 Word、Excel、Teams 裡運作。Google 則透過 Gemini 串聯 Android、Workspace 和 Search，把 AI 能力灌注到全球數十億使用者每天使用的工具中。兩者的策略本質相同：不需要使用者學習新工具，AI 就在你原本的工作流裡。這是通路優勢的極致發揮，當你已經擁有全球最大的企業軟體和消費者平台，AI Agent 只是在既有管道上加一層智慧層。

**Anthropic 的路線：從開發者生態往上長。** Anthropic 走的是截然不同的路徑。Claude Code 已經在開發者社群建立了「寫程式最強」的口碑，而新推出的 Cowork 和 Dispatch 則將 Claude 的能力從單一對話延伸到多 Agent 協作與任務調度。Cowork 讓使用者可以同時與多個 Claude 實例協同工作，Dispatch 則提供程式化的 Agent 編排能力，讓開發者能夠建構複雜的自動化工作流。

這條路線的邏輯是：先讓開發者用 Claude Code 建出各種垂直應用，再透過 Cowork 和 Dispatch 把這些應用串聯成企業級的 Agent 平台。Anthropic 沒有 Microsoft 的 Office 通路，也沒有 Google 的 Android 裝機量，但它選擇從「讓人造出工具」這個更底層的位置切入。如果 AI 時代的軟體越來越多是由 AI 自己寫出來的，那麼掌握「寫軟體的 AI」就等於掌握了上游。

**阿里巴巴的悟空則代表第三條路線：封閉生態的全面 AI 化。** 釘釘加上淘寶、天貓、支付寶形成的商業閉環，讓悟空可以在一個平台內完成從行銷、客服到財務的所有工作。這在中國市場有巨大優勢，但也意味著它的擴展性受限於阿里巴巴生態的邊界。

三條路線最終都在搶同一件事：成為企業 AI Agent 的預設入口。Microsoft 和 Google 靠既有通路，Anthropic 靠開發者生態和技術護城河，阿里巴巴靠封閉但完整的商業閉環。2026 年的下半場，勝負取決於誰能讓使用者養成「先開這個平台」的習慣。

---

阿里巴巴用一場發布會，同時展示了組織重組（ATH）、核心產品（悟空）、商業模式（Token 計費）、硬體佈局（釘釘 Real）和全球化野心。五個面向同時推進，這是一次完整的戰略宣言。

當 8 億使用者的工作平台被 AI 重寫底層，當一個人加上一群 Agent 就能運作一家公司，「辦公軟體」這個概念本身可能就要被重新定義了。

## 版本 2：精簡版（Threads）

釘釘 CEO 陳航：「今天把釘釘打碎，用 AI 重建，煉出『悟空』。」

阿里巴巴昨晚宣布成立 ATH（Alibaba Token Hub）事業群，今天就端出了「悟空」。這是一個企業級 AI Agent 平台，能夠 24 小時自主工作，定位是 AI 同事，而非聊天機器人。

以下三個重點值得關注。

悟空把釘釘的底層程式碼全部重寫為 CLI 架構，讓 AI Agent 可以原生操作上千項功能，不需要模擬人類點擊螢幕。8 億使用者的工作方式即將被重新定義。

同步推出「OPT 一人團隊」概念，首批覆蓋電商、跨境電商、法律、財稅等十大行業。一個人加上一群 AI Agent，就能完成過去整個團隊的工作。

計費方式也變了。不收月費，改用 Token 計費，用多少付多少，像水電費一樣。這讓中小企業也能用得起企業級 AI。

硬體方面，A1 Pro 錄音卡、Cleer H1 AI 耳機、企業級終端「釘釘 Real」同步亮相。釘釘是中國三大辦公平台中唯一走「軟硬整合」路線的。

悟空還宣布要進軍全球，後續將連接 Slack、微信等平台。配合淘寶、天貓、支付寶的 Skill 接入，它正在成為阿里巴巴 AI 能力的統一出口。

真正的戰場在「通路」。Microsoft 把 Copilot 塞進 365，Google 讓 Gemini 串聯 Workspace 和 Android，Anthropic 則靠 Claude Code 的開發者口碑加上 Cowork、Dispatch 的多 Agent 協作，從寫程式這個上游切入。阿里巴巴用釘釘加電商閉環打中國市場，但全球戰局遠未定論。

2026 年的企業 AI Agent 大戰，打的是誰能成為 AI 時代的預設入口。
