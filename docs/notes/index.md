# 筆記 (Notes)

> 快速記錄想法、學習心得與靈感捕捉

## 什麼是筆記？

筆記是最輕量級的內容類型，用於快速記錄想法、學習心得、讀書摘要或會議記錄。不需要嚴格的結構，重點是快速捕捉資訊。

## 與其他內容類型的區別

| 特性 | 筆記 | 文章 | 研究報告 |
|------|------|------|---------|
| 字數 | < 1,000 字 | 1,500-5,000 字 | 5,000+ 字 |
| 結構 | 彈性 | 有章節 | 嚴謹框架 |
| 完成度 | 可不完整 | 需完整 | 需完整 |
| 發布 | 通常不發布 | 可發布 | 可發布 |
| 用途 | 個人記錄 | 分享知識 | 深度分析 |

## 適用情境

- **學習筆記**：課程、書籍、文章的重點摘要
- **會議記錄**：會議要點、待辦事項、決策記錄
- **靈感捕捉**：突發想法、文章靈感、研究方向
- **讀書筆記**：書籍摘要、金句收集、個人反思
- **技術筆記**：指令備忘、設定紀錄、除錯經驗

## 檔案命名規範

```
YYYY-MM-DD-筆記主題.md

範例：
2026-01-12-docker-commands-cheatsheet.md
2026-01-15-meeting-notes-q1-planning.md
2026-01-20-book-notes-thinking-fast-slow.md
```

## 模板

使用 `/templates/note-template.md` 快速開始。

筆記模板結構簡單：
- 元資料
- 核心概念
- 要點記錄
- 個人思考
- 後續行動
