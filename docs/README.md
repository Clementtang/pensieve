# 文档目录

这个目录存放所有已完成并发布的文档。

## 子目录说明

### articles/
存放正式的文章，包括：
- 技术文章
- 深度分析
- 教程指南
- 经验总结

### notes/
存放简短的笔记和快速记录，包括：
- 学习笔记
- 读书笔记
- 会议记录
- 灵感记录

## 使用指南

1. **添加新文档：** 从 `drafts/` 移动已完成的文档到相应子目录
2. **文件命名：** 遵循 `YYYY-MM-DD-主题.md` 格式
3. **查找文档：** 使用 Git 搜索或按日期浏览

## 文档状态

所有这个目录下的文档都应该是"已发布"状态。
