# 企業研究 (Company Research)

> 系統性的企業商業分析與研究案例庫

## 關於本分類

「企業研究」是 Pensieve 中專門用於存放公司商業分析的主題分類。本分類的設計參考了哈佛商學院 (HBS)、MIT Sloan、柏克萊 Haas 等頂尖商學院的案例研究方法論，建立了一套標準化的分析框架。

## 研究框架特色

本框架整合了以下經典商業分析工具：

| 分析工具 | 來源/應用 | 用途 |
|---------|----------|------|
| 波特五力分析 | 哈佛商學院 | 產業競爭結構分析 |
| 商業模式畫布 | Osterwalder | 商業模式全貌解析 |
| SWOT 分析 | 經典策略工具 | 內外部環境評估 |
| 護城河分析 | 巴菲特/晨星 | 競爭優勢持久性評估 |
| 杜邦分析 | 財務分析 | 獲利能力拆解 |
| 安索夫矩陣 | 成長策略 | 成長途徑規劃 |

## 研究報告結構

每份企業研究報告包含十大部分：

```
1. 執行摘要 (Executive Summary)
   └── 快速掌握研究精華

2. 公司概況 (Company Overview)
   └── 基本資料、歷史沿革、使命願景

3. 產業分析 (Industry Analysis)
   └── 波特五力、產業趨勢、價值鏈

4. 商業模式分析 (Business Model Analysis)
   └── 商業模式畫布、營收模式、護城河

5. 競爭分析 (Competitive Analysis)
   └── 競爭格局、競爭者比較、策略定位

6. 財務分析 (Financial Analysis)
   └── 關鍵指標、成長分析、現金流

7. SWOT 分析
   └── 優勢、劣勢、機會、威脅、交叉策略

8. 管理團隊與治理 (Management & Governance)
   └── 管理層、董事會、股權結構

9. 風險分析 (Risk Analysis)
   └── 風險矩陣、風險清單、緩解措施

10. 成長策略與展望 (Growth Strategy & Outlook)
    └── 策略方向、情境分析、未來展望
```

## 檔案命名規則

企業研究報告遵循以下命名格式：

```
YYYY-MM-DD-company-name-research.md
```

**範例：**
- `2025-11-23-tsmc-research.md`
- `2025-11-23-apple-research.md`
- `2025-11-23-nvidia-research.md`

## 研究深度分級

| 等級 | 名稱 | 說明 | 預估時間 |
|------|------|------|---------|
| L1 | 初步研究 | 快速概覽，填寫基本資料與關鍵發現 | 2-4 小時 |
| L2 | 深度分析 | 完整填寫所有章節，深入分析 | 1-2 週 |
| L3 | 投資評估 | 加入詳細財務模型與估值分析 | 2-4 週 |

## 使用方式

### 建立新的企業研究

```bash
# 複製模板到企業研究目錄
cp templates/company-research-template.md docs/company-research/YYYY-MM-DD-company-name-research.md

# 開始編輯
vim docs/company-research/YYYY-MM-DD-company-name-research.md
```

### 請 Claude 協助

```
"請使用企業研究模板幫我建立一份關於 [公司名稱] 的研究報告"
"請幫我分析 [公司名稱] 的商業模式與競爭優勢"
"請根據企業研究框架評估 [公司名稱] 的投資價值"
```

## 品質檢查清單

完成企業研究後，請確認以下項目：

- [ ] 執行摘要能讓讀者在 3 分鐘內掌握重點
- [ ] 所有資料都標註來源與取得日期
- [ ] 財務數據為最新且正確
- [ ] 競爭分析包含至少 3 家主要競爭者
- [ ] SWOT 各項至少列出 3 點
- [ ] 風險分析考慮各種類型風險
- [ ] 結論有具體的評分與建議
- [ ] 附錄完整列出所有參考資料

## 研究倫理

- 確保資料來源可靠且可驗證
- 明確標註個人觀點與客觀事實
- 聲明研究限制與潛在偏見
- 不構成投資建議的免責聲明
