# 企業研究起始提示

## 使用時機
開始一個新的企業研究專案時使用。

---

## 提示詞模板

```
我想進行 [公司名稱] 的深度企業研究。

**研究目標：**
- [投資評估 / 競爭分析 / 產業研究 / 案例學習]

**預期產出：**
- 完整企業研究報告（參考 templates/company-research-template.md）
- 目標字數：[10,000-20,000] 字

**特別關注領域：**
1. [商業模式與競爭優勢]
2. [財務表現與估值]
3. [成長策略與風險]

**已知資訊：**
- [列出你已經知道的關於這間公司的資訊]

**研究步驟：**
1. 先進行網路搜尋，收集最新資訊
2. 依照 company-research-template.md 的結構組織內容
3. 確保符合 WRITING_GUIDE.md 的台灣繁體中文用語規範

請先進行初步搜尋，告訴我你找到的關鍵資訊，我們再決定研究重點。
```

---

## 變體

### 快速版（適合社群貼文）
```
我需要快速了解 [公司名稱] 的核心業務。

請搜尋並整理：
1. 公司基本資訊（成立時間、創辦人、總部）
2. 核心產品/服務
3. 商業模式
4. 最近的重大新聞（3 個月內）
5. 估值/融資狀況

輸出格式：bullet points，每點 1-2 句話。
```

### 更新版（已有研究基礎）
```
我之前已完成 [公司名稱] 的研究報告（見附件/檔案路徑）。

現在有新的發展：[描述新聞/事件]

請幫我：
1. 搜尋相關的最新資訊
2. 分析這對公司的影響
3. 建議報告中需要更新的章節
```
