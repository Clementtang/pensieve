# 研究提示詞模板

這個目錄存放與 Claude 協作研究時的標準化提示詞模板。

## 使用方式

1. 選擇適合的提示詞模板
2. 複製模板內容
3. 替換 `[佔位符]` 為實際資訊
4. 與 Claude 開始研究對話

---

## 可用模板

### 企業研究

- **company-research-start.md** - 企業研究起始提示
- **financial-analysis.md** - 財務分析提示
- **competitive-analysis.md** - 競爭分析提示

### 產業研究

- **industry-overview.md** - 產業概覽提示

### 通用

- **news-analysis.md** - 新聞事件分析提示
- **social-post.md** - 社群貼文生成提示

---

## 提示詞設計原則

1. **明確目標**：清楚說明期望的輸出格式和內容
2. **提供脈絡**：給予足夠的背景資訊
3. **指定規範**：引用 WRITING_GUIDE.md 確保風格一致
4. **分階段執行**：複雜任務分拆成多個步驟
