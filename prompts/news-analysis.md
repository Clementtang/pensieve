# 新聞事件分析提示

## 使用時機
分析重大商業新聞或產業事件時使用。

---

## 提示詞模板

```
我看到這則新聞：[貼上新聞標題或連結]

請幫我進行深度分析：

**分析框架：**
1. **事件摘要**：發生了什麼？關鍵數字？
2. **背景脈絡**：為什麼會發生？之前有什麼跡象？
3. **利害關係人**：誰受益？誰受損？
4. **產業影響**：對該產業的短期/長期影響？
5. **台灣觀點**：對台灣企業/投資人的啟示？

**輸出要求：**
- 符合 WRITING_GUIDE.md 的台灣繁體中文用語
- 避免「不是 X，而是 Y」句型
- 適合商業讀者的專業但易懂風格

請先搜尋相關背景資訊，再進行分析。
```

---

## 變體

### 併購分析
```
[公司A] 宣布併購/收購 [公司B]，交易金額 [金額]。

請分析：
1. 併購動機（買方為何要買？）
2. 估值合理性（這個價格合理嗎？）
3. 整合挑戰（可能遇到什麼問題？）
4. 競爭影響（對競爭對手的影響？）
5. 成功機率（歷史上類似併購的成功率？）

請搜尋雙方公司的背景資訊，以及類似併購案例。
```

### 融資分析
```
[公司名稱] 完成 [輪次] 融資，金額 [金額]，估值 [估值]。

請分析：
1. 估值變化（與上一輪比較）
2. 投資人背景（領投方的投資風格）
3. 資金用途（公司計劃怎麼用這筆錢？）
4. 市場信號（這對該產業代表什麼？）
5. 台灣連結（有無台灣相關機會？）
```
