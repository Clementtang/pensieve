## 使用方式

### 在手機（Claude Code Web）建立 Memo

```
我正在為 [主題] 進行 [文章/企業研究/議題研究] 的前期研究。

請幫我：
1. 搜尋關鍵資訊
2. 整理發現
3. 建立 Research Memo（參考 templates/research-memo-template.md）

保持 memo 在 1000 字以內。我稍後會在 CLI 繼續撰寫。
```

### 在電腦（Claude Code CLI）續寫

```
我有一份研究備忘錄在 drafts/memo-[topic].md。

請：
1. 讀取 memo
2. 讀取 templates/ 中對應的模板
3. 參照 WRITING_GUIDE.md 的標準
4. 撰寫完整 [文章/研究報告]

遵循台灣用語規範。
```
