# 模板目录

这个目录存放各种文章模板，用于快速开始新的写作。

## 可用模板

### 1. article-template.md
**用途：** 标准技术文章或长篇内容

**适用场景：**
- 技术文章
- 深度分析
- 经验总结
- 完整教程

**包含结构：**
- 元数据
- 引言
- 主要内容章节
- 总结和参考资料

---

### 2. note-template.md
**用途：** 快速记录笔记和要点

**适用场景：**
- 学习笔记
- 读书摘要
- 会议记录
- 灵感捕捉

**包含结构：**
- 核心概念
- 要点记录
- 个人思考
- 后续行动

---

### 3. tutorial-template.md
**用途：** 分步教程和操作指南

**适用场景：**
- 技术教程
- 操作指南
- 实践项目
- 学习路径

**包含结构：**
- 前置要求
- 分步说明
- 代码示例
- 故障排除
- 练习题

---

## 使用方法

### 方法 1：命令行复制

```bash
# 复制模板到草稿目录
cp templates/article-template.md drafts/2025-11-20-my-article.md

# 开始编辑
vim drafts/2025-11-20-my-article.md
```

### 方法 2：请 Claude 帮助

```
"请使用 article-template.md 模板帮我创建一篇关于 [主题] 的文章"
```

---

## 自定义模板

你可以根据需要创建自己的模板：

1. 创建新的 `.md` 文件
2. 定义你常用的结构
3. 保存到这个目录
4. 在使用时复制
