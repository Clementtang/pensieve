# 模板目錄

這個目錄存放各種文章模板，用於快速開始新的寫作。

## 可用模板

### 1. article-template.md
**用途：** 標準技術文章或長篇內容

**適用場景：**
- 技術文章
- 深度分析
- 經驗總結
- 完整教學

**包含結構：**
- 元資料
- 引言
- 主要內容章節
- 總結和參考資料

---

### 2. note-template.md
**用途：** 快速記錄筆記和要點

**適用場景：**
- 學習筆記
- 讀書摘要
- 會議記錄
- 靈感捕捉

**包含結構：**
- 核心概念
- 要點記錄
- 個人思考
- 後續行動

---

### 3. tutorial-template.md
**用途：** 分步教學和操作指南

**適用場景：**
- 技術教學
- 操作指南
- 實作專案
- 學習路徑

**包含結構：**
- 前置要求
- 分步說明
- 程式碼範例
- 故障排除
- 練習題

---

### 4. company-research-template.md
**用途：** 企業商業分析與研究報告

**適用場景：**
- 投資研究
- 競爭分析
- 產業研究
- 策略規劃
- 商業案例研究

**包含結構：**
- 執行摘要 (Executive Summary)
- 公司概況 (Company Overview)
- 產業分析 (Industry Analysis) - 波特五力
- 商業模式分析 (Business Model Analysis) - 商業模式畫布、護城河
- 競爭分析 (Competitive Analysis)
- 財務分析 (Financial Analysis) - 杜邦分析
- SWOT 分析與交叉策略矩陣
- 管理團隊與治理 (Management & Governance)
- 風險分析 (Risk Analysis)
- 成長策略與展望 (Growth Strategy & Outlook)

**參考框架：**
- 哈佛商學院 (HBS) 案例研究方法
- MIT Sloan 策略分析框架
- 柏克萊 Haas 商業分析模型

---

### 5. industry-research-template.md
**用途：** 產業/技術/市場深度研究報告（Topic Research）

**適用場景：**
- 產業分析（如：電子貨架標籤、AI 晶片）
- 技術趨勢（如：邊緣運算、量子計算）
- 市場研究（如：東南亞電商、台灣 SaaS）

**與 company-research 的差異：**
- 涵蓋整個產業鏈上中下游多家公司
- 著重產業結構與技術演進
- 滾動更新（季度/年度）

**包含結構（7 大章節）：**
1. 市場概況與規模
2. 產業價值鏈分析
3. 技術架構與演進
4. 競爭格局分析
5. 應用場景與案例
6. 商業模式分析
7. 未來展望

**品質標準：**
達到市調/顧問公司研究報告水準，數據支撐、參考資料完整

---

### 6. topic-research-template.md
**用途：** 議題研究與政策分析

**適用場景：**
- 政策議題
- 科技倫理
- 社會趨勢
- 哲學探討

**包含結構：**
- 議題界定
- 歷史脈絡
- 利害關係人分析
- 多維度分析（PESTEL、系統思考、倫理框架）
- 論證分析
- 情境規劃
- 解決方案與建議

---

## 使用方法

### 方法 1：指令行複製

```bash
# 複製模板到草稿目錄
cp templates/article-template.md drafts/2025-11-20-my-article.md

# 開始編輯
vim drafts/2025-11-20-my-article.md
```

### 方法 2：請 Claude 幫助

```
"請使用 article-template.md 模板幫我建立一篇關於 [主題] 的文章"
```

---

## 自訂模板

你可以根據需要建立自己的模板：

1. 建立新的 `.md` 檔案
2. 定義你常用的結構
3. 儲存到這個目錄
4. 在使用時複製
