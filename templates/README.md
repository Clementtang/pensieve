# 模板目錄

這個目錄存放各種文章模板，用於快速開始新的寫作。

---

## 快速選擇指南

### 依內容類型選擇

```
你要寫什麼？
│
├─ 快速記錄（< 1,000 字）
│  ├─ 學習/讀書/會議筆記 → note-template.md
│  └─ 行動端研究備忘 → research-memo-template.md
│
├─ 文章（1,500-5,000 字）
│  ├─ 一般主題分析 → article-template.md
│  └─ 步驟式教學 → tutorial-template.md
│
├─ 企業研究（5,000-20,000 字）
│  └─ 單一公司深度分析 → company-research-template.md
│
└─ 產業/議題研究
   ├─ 產業/技術/市場研究 → industry-research-template.md
   └─ 複雜議題/政策分析
      ├─ 1-2 週完成 → topic-research-lite-template.md
      └─ 4-8 週完成 → topic-research-template.md
```

### 依時間預算選擇

| 時間 | 推薦模板 | 目標字數 |
|------|---------|---------|
| 30 分鐘 | note-template | < 500 字 |
| 1-2 小時 | research-memo-template | 500-1,000 字 |
| 半天-1 天 | article-template | 1,500-3,000 字 |
| 2-3 天 | tutorial-template | 2,000-5,000 字 |
| 1-2 週 | topic-research-lite-template | 3,000-8,000 字 |
| 2-4 週 | company-research / industry-research | 10,000-20,000 字 |
| 4-8 週 | topic-research-template | 15,000-50,000 字 |

## 可用模板

### 1. article-template.md
**用途：** 標準技術文章或長篇內容

**適用場景：**
- 技術文章
- 深度分析
- 經驗總結
- 完整教學

**包含結構：**
- 元資料
- 引言
- 主要內容章節
- 總結和參考資料

---

### 2. note-template.md
**用途：** 快速記錄筆記和要點

**適用場景：**
- 學習筆記
- 讀書摘要
- 會議記錄
- 靈感捕捉

**包含結構：**
- 核心概念
- 要點記錄
- 個人思考
- 後續行動

---

### 3. tutorial-template.md
**用途：** 分步教學和操作指南

**適用場景：**
- 技術教學
- 操作指南
- 實作專案
- 學習路徑

**包含結構：**
- 前置要求
- 分步說明
- 程式碼範例
- 故障排除
- 練習題

---

### 4. company-research-template.md
**用途：** 企業商業分析與研究報告

**適用場景：**
- 投資研究
- 競爭分析
- 產業研究
- 策略規劃
- 商業案例研究

**包含結構：**
- 執行摘要 (Executive Summary)
- 公司概況 (Company Overview)
- 產業分析 (Industry Analysis) - 波特五力
- 商業模式分析 (Business Model Analysis) - 商業模式畫布、護城河
- 競爭分析 (Competitive Analysis)
- 財務分析 (Financial Analysis) - 杜邦分析
- SWOT 分析與交叉策略矩陣
- 管理團隊與治理 (Management & Governance)
- 風險分析 (Risk Analysis)
- 成長策略與展望 (Growth Strategy & Outlook)

**參考框架：**
- 哈佛商學院 (HBS) 案例研究方法
- MIT Sloan 策略分析框架
- 柏克萊 Haas 商業分析模型

---

### 5. industry-research-template.md
**用途：** 產業/技術/市場深度研究報告（Topic Research）

**適用場景：**
- 產業分析（如：電子貨架標籤、AI 晶片）
- 技術趨勢（如：邊緣運算、量子計算）
- 市場研究（如：東南亞電商、台灣 SaaS）

**與 company-research 的差異：**
- 涵蓋整個產業鏈上中下游多家公司
- 著重產業結構與技術演進
- 滾動更新（季度/年度）

**包含結構（7 大章節）：**
1. 市場概況與規模
2. 產業價值鏈分析
3. 技術架構與演進
4. 競爭格局分析
5. 應用場景與案例
6. 商業模式分析
7. 未來展望

**品質標準：**
達到市調/顧問公司研究報告水準，數據支撐、參考資料完整

---

### 6. topic-research-template.md (Standard)
**用途：** 議題研究與政策分析（完整版）

**適用場景：**
- 深度學術研究
- 系統性政策分析
- 複雜議題的全面探討
- 需要 4-8 週完成的研究

**包含結構（10 部分）：**
- 執行摘要
- 議題界定
- 歷史脈絡
- 利害關係人分析
- 多維度分析（PESTEL、系統思考、倫理框架）
- 論證分析
- 比較分析
- 情境規劃
- 解決方案與建議
- 研究限制與反思

**目標字數：** 15,000-50,000 字

---

### 7. topic-research-lite-template.md (Lite)
**用途：** 議題研究（簡化版）

**適用場景：**
- 快節奏的議題探討
- 初步研究或前期探索
- 時間有限（1-2 週）的專案
- 中等複雜度的議題

**包含結構（5 部分）：**
- 執行摘要
- 議題界定
- 脈絡與現狀（合併歷史、利害關係人、論點分析）
- 分析與前景（簡化多維度分析與情境規劃）
- 解決方案與結論

**目標字數：** 3,000-8,000 字

---

### 📋 議題研究模板選擇指南

選擇適合的版本：

```
你有多少時間？
├── 1-2 週 → 使用 Lite 版本
└── 4-8 週 → 使用 Standard 版本

議題複雜度？
├── 簡單/中等 → 使用 Lite 版本
└── 複雜/多面向 → 使用 Standard 版本

研究目的？
├── 初探/快速理解 → 使用 Lite 版本
├── 深度分析/學術發表 → 使用 Standard 版本
└── 決策支援 → 根據時間選擇
```

---

### 8. research-memo-template.md
**用途：** Web↔CLI 工作流的研究備忘錄

**適用場景：**
- 手機端前期研究
- 跨平台工作流交接
- 快速捕捉研究發現

**目標字數：** 1,000 字以內

## 使用方法

### 方法 1：指令行複製

```bash
# 複製模板到草稿目錄
cp templates/article-template.md drafts/2025-11-20-my-article.md

# 開始編輯
vim drafts/2025-11-20-my-article.md
```

### 方法 2：請 Claude 幫助

```
"請使用 article-template.md 模板幫我建立一篇關於 [主題] 的文章"
```

## 自訂模板

你可以根據需要建立自己的模板：

1. 建立新的 `.md` 檔案
2. 定義你常用的結構
3. 儲存到這個目錄
4. 在使用時複製
