# 模板目錄

這個目錄存放各種文章模板，用於快速開始新的寫作。

## 可用模板

### 1. article-template.md
**用途：** 標準技術文章或長篇內容

**適用場景：**
- 技術文章
- 深度分析
- 經驗總結
- 完整教學

**包含結構：**
- 元資料
- 引言
- 主要內容章節
- 總結和參考資料

---

### 2. note-template.md
**用途：** 快速記錄筆記和要點

**適用場景：**
- 學習筆記
- 讀書摘要
- 會議記錄
- 靈感捕捉

**包含結構：**
- 核心概念
- 要點記錄
- 個人思考
- 後續行動

---

### 3. tutorial-template.md
**用途：** 分步教學和操作指南

**適用場景：**
- 技術教學
- 操作指南
- 實作專案
- 學習路徑

**包含結構：**
- 前置要求
- 分步說明
- 程式碼範例
- 故障排除
- 練習題

---

## 使用方法

### 方法 1：指令行複製

```bash
# 複製模板到草稿目錄
cp templates/article-template.md drafts/2025-11-20-my-article.md

# 開始編輯
vim drafts/2025-11-20-my-article.md
```

### 方法 2：請 Claude 幫助

```
"請使用 article-template.md 模板幫我建立一篇關於 [主題] 的文章"
```

---

## 自訂模板

你可以根據需要建立自己的模板：

1. 建立新的 `.md` 檔案
2. 定義你常用的結構
3. 儲存到這個目錄
4. 在使用時複製
